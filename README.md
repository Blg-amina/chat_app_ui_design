# 📱 Chat App UI Design

A simple **Flutter** project that showcases a clean and modern **chat application user interface**.

## 🚀 Features
- Beautiful and responsive UI
- Chat list screen
- Chat detail screen
- Custom colors and typography
- Organized Flutter widgets

## 🛠️ Technologies Used
